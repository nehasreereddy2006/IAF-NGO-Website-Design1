# 🌱 NGO Website Feature Highlights
### IAF Internship — Task 4 | UI/UX Design

Designed by **Neha Reddy** | B.Tech CSE | RGUKT Basar  
InAmigos Foundation Open Innovation Internship — June 2025

---

## 📋 About This Project
This project identifies and visually presents **5 key features** that every NGO website should have, with a focus on InAmigos Foundation's website improvement.

---

## ✨ Features Designed

| # | Feature | Purpose |
|---|---------|---------|
| 01 | 🏠 Hero Banner | Mission statement + primary CTA |
| 02 | 💛 Donation Section | Preset amounts + fund progress bar |
| 03 | 📝 Volunteer Sign-Up Form | Community engagement |
| 04 | 📊 Impact Showcase & Testimonials | Build trust with numbers + stories |
| 05 | 🎯 Active Campaigns | Create urgency + drive participation |
